télécharger ici - ANTI-CONSUMERISM-WEB

“Anti-consumerism web” est une extension web à installer sur votre navigateur. Pour cela, suivez les indications listées ci-dessous :

  1. téléchargez le dossier “ANTI-CONSUMERISM-WEB” zip
  2. Décompresser ce fichier sur votre bureau
  3. Rendez-vous sur votre navigateur et ouvrez l’onglet des extensions → chrome://extensions/
  4. Activez le mode “développeur” en haut à droite
  5. Cliquez sur “Load Unpacked” et sélectionnez le dossier téléchargé plus tôt “ANTI-CONSUMERISM-WEB”
  6. Votre navigateur possède maintenant l’extension, vous ne pouvez plus consommer sur le web
